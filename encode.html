Eenvoudige steganografie code red = 2

  • encoder

  • dit script maakt 2 canvas elementen
  • canvas 1 voor de geheime tekst boodschap (canvas )
  • canvas 2 voor het originele image
  • de canvassen krijgen dezelfde afmetingen
  • de encoder loopt door canvas 1 en test op zwarte pixels. Een zwart pixel betekent onderdeel van de tekst boodschap
    als er een witte pixel is dan is dat geen onderdeel van de boodschap.
    Geen onderdeel van de boodschap =>in het originele image canvas2 mag de rode RGB NIET op 2 eindigen.
    Wel onderdeel van de boodschap =>in het originele image canvas2 MOET de rode RGB op 2 eindigen

  • decoder

  • de decoder loopt door het image met de boodschap
  • alle pixels waarvan de rode RGB NIET op 2 eindigt worden onderdrukt
